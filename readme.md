# ChessYourDream

เว็บแอปพลิเคชันสำหรับจดบันทึกประจำวันและให้คะแนนความรู้สึกตัวเอง (Mood Tracker) พัฒนาเพื่อฝึกฝนทักษะ Python และ Web Development

##  ฟีเจอร์หลัก
*  **ปฏิทิน:** ดูภาพรวมคะแนนอารมณ์ในแต่ละเดือน
*  **ให้คะแนน:** ให้คะแนนความรู้สึกรายวัน (1-5)
*  **จดบันทึก:** เขียนไดอารี่สั้นๆ
*  **รูปภาพ:** แนบรูปความทรงจำในแต่ละวันได้

##  ใช้
* **Python** (Flask Framework)
* **Database:** PostgreSQL
* **Frontend:** HTML, CSS
